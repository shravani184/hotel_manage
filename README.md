A full-stack Hotel Booking System built using React (Vite) for the frontend and MySQL for the database.  
The system allows users to browse rooms, book rooms, manage profiles, and provides an admin dashboard for management.

Features

User
- User Signup & Login
- View available rooms
- Book rooms
- View booking details
- Profile management

Admin
- Admin dashboard
- View all bookings
- Manage rooms and users

---

Tech Stack

Frontend
- React (Vite)
- Tailwind CSS
- JavaScript (ES6)
- Axios / Fetch API

Backend / Database
- MySQL

Tools
- VS Code
- MySQL Server / MySQL Workbench
- Git & GitHub
- Node.js & npm
